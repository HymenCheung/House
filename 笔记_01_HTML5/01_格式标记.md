### 段落标记 \*

> 标记名：p
> 
> 标记类型：双标记
> 
> 常用属性：

| 属性名   | 属性名解释 | 属性值    | 属性值解释 | 默认  |
| ----- | ----- | ------ | ----- | --- |
| align | 水平对齐  | left   | 左对齐   | 是   |
|       |       | center | 居中对齐  |     |
|       |       | right  | 右对齐   |     |

### 换行标记

> 标记名：br
> 
> 标记类型：单标记

### 分割线标记

> 标记名：hr
> 
> 标记类型：单标记
> 
> 常用属性：

| 属性名   | 属性名解释 | 属性值    | 属性值解释 | 默认  |
| ----- | ----- | ------ | ----- | --- |
| align | 水平对齐  | left   | 左对齐   |     |
|       |       | center | 居中对齐  | 是   |
|       |       | right  | 右对齐   |     |
| width | 分割线长度 | 数值+单位  | 分割线长度 |     |
| size  | 分割线宽度 | 数值+单位  | 分割线宽度 |     |
| color | 分割线颜色 | 颜色     | 分割线颜色 |     |

> 标记名：font
> 
> 标记类型：双标记

| 属性名   | 属性名解释 | 属性值   | 属性值解释     | 默认  |

| size  | 字体大小  | +/-倍数 | 字体放大/缩小倍数 |     |

### 超链接标记\*

> 标记名：a
> 
> 标记类型：双标记
> 
> 常用属性：

| 属性名    | 属性名解释 | 属性值    | 属性值解释 | 默认  |
| ------ | ----- | ------ | ----- | --- |
| href   | 超链接路径 | url    | 超链接路径 |     |
| target | 目标窗体  | _self  | 当前窗体  | 是   |
|        |       | _blank | 新窗体   |     |
