## 节段标记

### 列表标记

#### 有序列表标记

> 标记名：ol
> 
> 标记类型：双标记

#### 无序列表标记

> 标记名：ul
> 
> 标记类型：双标记

#### 列表项标记

> 标记名：li
> 
> 标记类型：双标记

### 块级元素\*

> 标记名：div
> 
> 标记类型：双标记
> 
> 派生标记：header、nav、aside、section、article、footer
> 
> 常用属性：

| 属性名   | 属性名解释 | 属性值    | 属性值解释 | 默认  |
| ----- | ----- | ------ | ----- | --- |
| align | 水平对齐  | left   | 左对齐   | 是   |
|       |       | center | 居中对齐  |     |
|       |       | right  | 右对齐   |     |

### 行内元素\*

> 标记名：span
> 
> 标记类型：双标记
